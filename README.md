# Gem Wallet - Solana Address Publishing

This document outlines the production configuration for **Gem Wallet** based on [Solana Mobile dApp Publishing](https://github.com/danielmosesochefije54-netizen/solana-address/dapp-publishing). al production setup.

## Prerequisites

- **Keypair Location**: Include `/solana-address/wallet/keypair.json` that's used for signing application.
- **APK Location**: Latest build should be under `files/app-release.apk`
